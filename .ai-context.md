# Mecalin Project Context

This is a GTK4 application written in Rust using the Adwaita design system and Meson build system.

## Project Structure
- `src/` - Rust source code
- `resources/` - UI files and other resources embedded via GResource
- `Cargo.toml` - Rust dependencies and project metadata
- `meson.build` - Meson build configuration
- `build.rs` - Compiles GResources at build time

## Key Technologies
- **GTK4** - Modern cross-platform GUI toolkit
- **libadwaita** - GNOME's design system and widgets
- **GResource** - Embeds UI files and assets into the binary
- **GTK Builder** - Declarative UI definition using XML templates
- **Meson** - Build system for compilation and packaging

## Building and Running

### Development (Cargo)
```bash
cargo run
```

### Production (Meson)
```bash
meson setup builddir
meson compile -C builddir
./builddir/mecalin
```

## Dependencies
- GTK4 4.10+
- libadwaita 1.5+
- Rust toolchain
- Meson build system
- pkg-config

## Architecture
The application uses GTK's composite template system where UI is defined in XML files and bound to Rust structs using the `#[derive(CompositeTemplate)]` pattern.
